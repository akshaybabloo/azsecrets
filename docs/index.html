Welcome to Secrets’s documentation!¶

Secrets can be used to set your environment variables that’s stored in Azure KeyVault, it can also be used as a package to retrieve the secrets.

It can be used in two ways:

  • As a Command Line Interface
    • Setting up Shell before using Secrets
    • Comma Separated Names
    • Arguments and Commands
  • As a Module
    • Get a Single Secret
    • Get All Secrets
    • Package Contents
